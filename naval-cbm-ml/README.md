## Overview

This project builds a complete end-to-end machine-learning pipeline for **predictive condition-based maintenance (CBM)** of naval gas turbine propulsion systems. Given 16 real-time sensor readings from a CODLAG propulsion plant, four models are trained to simultaneously predict two degradation coefficients — **GT Compressor Decay** and **GT Turbine Decay** — before failures occur.

The best model (XGBoost) achieves **R² = 0.9928** with RMSE < 10⁻³, well within operational tolerance for early-warning maintenance systems.

## Dataset

**UCI Naval Propulsion Plants** — [UCI ML Repository](https://archive.ics.uci.edu/dataset/316/condition+based+maintenance+of+naval+propulsion+plants)

- 11,934 samples · 16 sensor features · 2 continuous targets
- Simulation-based (CODLAG plant — diesel-electric + gas turbine)
- No missing values

Place `data.txt` inside the `data/` folder before running:

```
naval-cbm-ml/

    └── data.txt      ← download from UCI
```

## Pipeline Summary

### 1. EDA
- Descriptive statistics (mean, std, skewness, kurtosis)
- Feature histograms and box plots
- Pearson correlation heatmap
- Pair plots for top-5 correlated features

### 2. Preprocessing
- Zero-variance feature removal (`T1`, `P1` are constants)
- IQR outlier detection → retained (valid operating states)
- `StandardScaler` fit only on train set (no leakage)

### 3. Feature Engineering
- **Compressor Pressure Ratio**: `Πc = P2 / P1`
- **Turbine Power Proxy**: `WT = GT_Shaft_Torque × GT_Rate_Revolution`
- **Operating Mode**: categorical (Low/Medium/High) from Lever Position

### 4. Models
| Model | Family | Tuning |
|---|---|---|
| Ridge Regression | Linear | `RidgeCV` — 5-fold CV over α grid |
| Random Forest | Ensemble (Bagging) | `GridSearchCV` — n_estimators, max_depth, features |
| XGBoost | Ensemble (Boosting) | `GridSearchCV` — depth, lr, subsample, regularisation |
| ANN (Keras) | Neural | EarlyStopping + ReduceLROnPlateau; 256→128→64→2 |

### 5. Evaluation
- RMSE, MAE, R², Adjusted R² (per-target and averaged)
- Overfitting diagnosis: Train / Val / Test R² comparison
- Wilcoxon signed-rank test for statistical significance
- Learning curves, residual plots, Q-Q plots, predicted-vs-actual

## Engineering Interpretation

- **Compressor Decay** operating range: [0.95, 1.0] → XGBoost RMSE ≈ **1.90% of range**
- **Turbine Decay** operating range: [0.975, 1.0] → XGBoost RMSE ≈ **3.07% of range**

Both are below the 5% threshold considered acceptable for early fault-warning systems in marine propulsion CBM.

## Limitations

- Dataset is simulation-generated (not from operational vessels)
- Real-world sensor noise may not be fully represented
- No temporal/sequence modelling (LSTM/Transformer future work)
- Cross-vessel generalisation not evaluated
